## FavorAutoAccept

Press **E** once on a Freerunner Post Board or Lockbox, the addon skips both dialog screens and accepts/completes the quest automatically.

### Slash Commands
```
/faa on | off | toggle
```

### Features
- One-keypress auto-accept from Freerunner Post Boards
- One-keypress auto-complete at Freerunners Post Lockbox

## Installation

1. Download the latest release
2. Copy `FavorAutoAccept/` into:
   ```
   Documents/Elder Scrolls Online/live/AddOns/
   ```
3. Restart ESO or `/reloadui`

## Compatibility

- **ESO API Version:** 101041 (Gold Road)
- **Dependencies:** None
- **Conflicts:** None known
